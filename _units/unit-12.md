# Unit 12

## Reflection
### What?
Unit 12 connected machine learning to Industry 4.0 and focused on how AI, automation, and data-driven systems are changing real industrial environments. The unit covered topics such as deployment in production settings, digital twins, self-supervised learning, edge AI, Neural Architecture Search, and the wider social and sustainability implications of these developments.
### So what?
This unit was important because it placed machine learning in a broader real-world context. It showed that successful ML is not only about building accurate models, but also about reliability, maintenance, scalability, and responsible deployment. I also found the discussion of digital twins and edge AI especially useful because they demonstrated how machine learning can support monitoring and decision-making in dynamic environments rather than remaining only an academic exercise.
### Now what?
Going forward, I want to think more critically about how machine learning systems operate after deployment, including their practical, ethical, and organizational impact. Unit 12 reinforced that future ML work should be judged not only by technical performance, but also by how well it supports sustainable and human-centred systems.
