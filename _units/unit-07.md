# Unit 7

## Reflection
### What?
Unit 7 introduced artificial neural networks and marked the move from classical machine learning into neural networks. It covered the biological inspiration behind ANNs, the perceptron as a weighted threshold classifier, weight updates through error reduction, the AND example, and the limitation of single layer models on XOR, which led into multi layer perceptrons and sigmoid activation.
### So what?
This unit helped me understand that a perceptron is useful for simple linear problems, but not for non linear ones. The AND activity showed how repeated weight updates can reach correct classification, while the XOR example made it clear why hidden layers and non linear activation functions are necessary. Running the exercises made the theory more concrete, especially when changing weights and inputs, comparing learning rates, and fixing the multi layer notebook so the error could decrease properly during training.
### Now what?
Going forward, I want to be more confident in explaining what neural networks are doing internally, not just using them as black box models. Unit 7 gave me a stronger foundation for later deep learning topics by showing why architecture, activation, and implementation details matter.
