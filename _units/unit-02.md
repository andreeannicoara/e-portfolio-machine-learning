# Unit 2

## Reflection
### What?
Unit 2 focused on exploratory data analysis and preprocessing. Through the Auto-mpg exercise, I examined missing values, skewness, kurtosis, correlations, and scatter plots, and documented how categorical values were encoded. The seminar also reinforced that EDA is a necessary stage before modelling, not an optional extra.
### So what?
This unit helped me understand that reliable modelling depends on understanding the dataset first. In the Auto-mpg data, I found that horsepower had missing values coded as “?”, that several variables were non-normal, and that mpg had clear negative relationships with weight, horsepower, and displacement. This showed me how EDA can reveal issues that would affect model choice, preprocessing, and interpretation if ignored.
### Now what?
Going forward, I want to treat EDA as a standard first step in every machine learning task. I also want to become more systematic in documenting assumptions, handling missingness, and distinguishing true numeric variables from encoded categories before moving into modelling.
