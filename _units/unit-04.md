# Unit 4

## Reflection
### What?
Unit 4 focused on applying linear regression in scikit-learn through a practical modelling workflow. I worked on preprocessing, fitting, and evaluating regression models, and completed the seminar exercise using global population and GDP datasets to test correlation and regression between mean population and mean GDP per capita.
### So what?
This unit showed me that running a model is not enough on its own. In the seminar task, the correlation between mean population and mean GDP per capita was very weak, r = -0.0995, and the regression model had almost no explanatory power, with R² = 0.0028. That made it clear that a variable can be easy to model in code but still be a poor predictor in practice. It also reinforced that regression outputs must be interpreted carefully and not treated as evidence of causality.
### Now what?
Going forward, I want to focus more on variable choice, evaluation metrics, and interpretation boundaries before claiming that a model is useful. Unit 4 helped me see that a sound machine learning workflow depends as much on critical judgement as on technical implementation.
