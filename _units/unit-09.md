# Unit 9

## Reflection
### What?
Unit 9 introduced Convolutional Neural Networks as a core deep learning approach for computer vision. The unit focused on convolution, filters, feature maps, pooling, and how learned image features are turned into final class predictions. I also completed the CNN activity by testing different input images in the provided notebook and comparing predicted labels with the true labels.
### So what?
This unit helped me understand that CNNs do not treat images as just large tables of pixels. Instead, they learn patterns such as edges, textures, and shapes across layers, which makes them more suitable for image classification tasks. The practical activity made this clearer because changing the input image and checking the predictions showed how the trained model behaved on individual cases. It also reinforced that accuracy alone is not enough, especially when CNNs are used in sensitive contexts such as facial recognition and surveillance.
### Now what?
Going forward, I want to pay more attention to how deep learning models make predictions, not just whether they are correct. Unit 9 showed me that understanding feature learning, model behavior, and ethical limits is important for using CNNs responsibly.
