# Unit 3

## Reflection
### What?
Unit 3 focused on correlation and regression, moving from describing relationships between variables to modelling them for prediction. I worked through notebooks on covariance, Pearson correlation, simple linear regression, multiple linear regression, and polynomial regression.
### So what?
This unit helped me understand that correlation is useful for identifying patterns, but it does not establish causation and only captures certain types of relationships. The practical exercises made this clearer. Increasing noise reduced Pearson’s r from 0.888 to 0.446, while adding a single outlier changed a negative correlation into a positive one and reversed the regression line. This showed me how sensitive correlation and regression can be to data quality, outliers, and modelling assumptions.
### Now what?
Going forward, I want to be more careful about checking assumptions before interpreting regression outputs. I also want to justify model choice more critically, especially when relationships may be non linear or distorted by outliers, rather than treating fitted models as automatically reliable.
